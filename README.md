# Node.js Supply Chain Security Guide

> Modern hardening guide for Node.js applications using PNPM v11, secure CI/CD and dependency protection strategies.

## Features

- PNPM v11 security configuration
- Hardened GitHub Actions
- Secure CI/CD examples
- Enterprise recommendations
- Dependency protection
- Incident response practices

## Quick Start

```bash
npm install -g pnpm
pnpm install
```

## Recommended Stack

- Node.js LTS
- PNPM v11+
- GitHub Actions Hardened
- Dependabot
- Snyk / Trivy / Semgrep
